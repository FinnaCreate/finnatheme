== Description ==

Finna is a blank canvas for your ideas and it makes the tailwindcss your best brush.

Take it for a spin! See how Finna elevates your portfolio, business website, or personal blog.

== Installation manually ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on the 'Upload Theme' button.
3. Choose the finna.1.0.zip file.
4. Click Install Now
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to your taste.

== Installation from wordpress ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Finna in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to your taste.

== For local development ==
1. Navigate to 'inc/header-fucntions.php'
2. Change the url for browser-sync in the finna_enqueue_scripts() function from 'wordpress.test' to your domain
2. cd to the theme folder and run 'npm install' and then npm run watch-sync

== Privacy ==
Finna uses LocalStorage to save the setting when Dark Mode support is turned on or off.
LocalStorage is necessary for the setting to work and is only used when a user clicks on the Dark Mode button.
No data is saved in the database or transferred.
